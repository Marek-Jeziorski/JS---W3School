Using splice() to Remove Elements

- first parameter (0) defines the position where new elements should be added (spliced in).
- second parameter (1) defines how many elements should be removed.
- rest of the parameters are omitted. No new elements will be added.

B) slice() method
- slices out a piece of an array.
- can take two arguments like slice(1, 3).
- then selects elements from the start argument, and up to (but not including) the end argument.

5 - Automatic toString()

JavaScript automatically converts an array to a comma separated string when a primitive value is expected.
This is always the case when you try to output an array.

.inner.HTML = arrayName.toString()
is same as
.innerHTML = arrayName

! All JavaScript objects have a toString() method.
